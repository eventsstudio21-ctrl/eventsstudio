## ğŸ†˜ Troubleshooting

**Video not playing?**
- Ensure MP4 is in same directory
- Check file path is correct
- Use H.264 codec
- File size under 50MB recommended

**Images not showing?**
- Verify file paths (absolute vs relative)
- Check image permissions
- Use modern formats (JPG, PNG, WebP)

**Form not submitting?**
- Need backend handler (PHP, Node.js, etc.)
- Use Formspree or similar service
- Add email integration

**Mobile menu not working?**
- Check viewport meta tag
- Test on actual mobile device
- Verify JavaScript is enabled
